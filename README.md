# FX Trade System

AIを搭載した高度なFX自動取引システム。

## 各ファイルの説明

- `01_データ収集.py`: 価格データの取得
- `02_AI学習.py`: 取引モデルのバックテストと学習
- `03_本番トレード.py`: 本番環境での取引実行
- `04_デモシミュレーション.py`: デモ口座での検証
- `05_夜間学習_進化.py`: モデルの定期更新
- `06_完全自動_無限進化.py`: 自動化された進化サイクル
- `07_永続進化_完全版.py`: システムの核となる進化ロジック
- `08_鉄壁の守り_完全版.py`: リスク管理と防御機能
- `09_究極のチューニング.py`: ハイパーパラメータの最適化
- `10_実戦強化_堅牢版.py`: 実戦向けの強化ロジック
- `11_実戦運用_完成形.py`: 最終的な統合運用スクリプト
- `config.py`: システム設定

## 使い方

1. `config.py` に OANDA API または MT5 の情報を入力してください。
2. 必要に応じて `01_データ収集.py` から順に実行してください。
